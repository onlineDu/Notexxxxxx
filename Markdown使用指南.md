# Markdown使用指南

## 1.多级标题

总共有六级标题,分别为: 

```
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题 
###### 六级标题 
```

PS：#和标题名称之间有一个空格

## 2.列表

### 2.1无序列表

 ```
  - 列表内容1
  
  + 列表内容2
  
  * 列表内容3
 ```


### 2.2有序列表
 ```
   1. 列表内容
   2. 列表内容
 ```

  ## 3.引用
```
> this is a test
> 引用的内容
```

## 4.字体设置

粗体：**需要加粗的文字**

斜体：*需要斜体的问题*

加粗并斜体：*** 需要加粗和斜体的文字***

##5.分割线

***或---代表分割线
